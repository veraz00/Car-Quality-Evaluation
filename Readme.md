# Car Quality Evaluation 

## Basics 

Based on [Kaggle: Car Evaluation Data Set](https://www.kaggle.com/datasets/elikplim/car-evaluation-data-set), I build a web app to deploy [Random Forest Classifier](https://www.kaggle.com/code/prashant111/random-forest-classifier-tutorial) built by [Prashant Banerjee](https://www.kaggle.com/prashant111)
